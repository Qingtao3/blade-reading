# Example Prompts

Use these prompts to trigger `blade-reading`.

## 1. Personal 30-day plan

```text
我是大一学生，想提升判断力、商业理解和表达能力，也想提升自媒体内容能力。不要鸡汤，不要堆名著，也不要推荐现阶段读不动的硬书。帮我做一份 30 天刃读计划。
```

## 2. Candidate list triage

```text
帮我筛这批候选书：原则、纳瓦尔宝典、穷查理宝典、鞋狗、乔布斯传、滚雪球、置身事内、经济学原理、思考快与慢、深度工作、刻意练习、认知觉醒、被讨厌的勇气、活着、局外人、悉达多、沉思录、影响力、娱乐至死、金字塔原理。

请告诉我哪些值得精读，哪些普通通读，哪些 AI 总结即可，哪些暂不优先。
```

## 3. Business books without hype

```text
我想读商业书，但不想看财富心法、商业认知、成功学。请帮我筛几本能理解真实交易、现金流、组织、风险、博弈和执行的书，并说明每本书的不可压缩价值。
```

## 4. AI tools and creator workflow

```text
我最近想学 AI 工具、Prompt、剪映和自媒体运营。请帮我判断哪些内容应该直接用 AI + 实操解决，哪些书有长期阅读价值。
```

## 5. Shelf cleanup

```text
下面是我的待读清单，请帮我整理成主分类和副标签，并说明哪些书暂不优先：

活着、百年孤独、卡拉马佐夫兄弟、原则、认知觉醒、从 0 到 1、创新者的窘境、邓小平时代、乡土中国、金钱心理学、Python 编程快速上手。
```
