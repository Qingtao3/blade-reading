# blade-reading

**中文**：AI 时代的狠筛书 skill。它根据使用者真实处境和人生问题，判断哪些书值得亲自读，哪些只需 AI 总结，并输出评分、不可压缩价值、阅读方式、分类和 30 天阅读顺序。

**English**: A sharp book-filtering skill for the AI era. It decides which books deserve personal reading, which are enough to summarize with AI, and produces scores, irreducible value, reading modes, categories, and a 30-day reading order.

## Boundary

**中文**：本 skill 不读取私有数据，不要求 API Key，不创建或修改微信读书书单。用户可以粘贴书架、笔记或候选书清单，skill 只基于文本生成筛书报告。

**English**: This skill does not read private data, require API keys, or create/modify WeRead booklists. Users may paste shelves, notes, or candidate lists; the skill only generates a text-based filtering report.
